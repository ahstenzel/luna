## luna
 Simple game framework based on Raylib.
 Includes the base `raylib` library, the `raygui` and `rres` modules, as well as extra wrappers for more abstract functionality.

 NOTE: This was done as an experiment in higher-level framework design (i.e. collision handling, object management, resource caching, etc) while letting existing solutions handle functions like rendering & file handling. This project has been supersceded by MARS, which is an attempt to roll a game framework from scratch as much as is practical. It has been archived and will likely not be updated in the future.
